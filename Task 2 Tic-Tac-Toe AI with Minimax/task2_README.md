# Task 2 — Tic-Tac-Toe AI

**CodSoft AI Internship**

## Overview
An AI agent that plays Tic-Tac-Toe against a human player using the **Minimax algorithm**. The AI is unbeatable — it always plays the optimal move. At best, you can draw against it!

## How it Works
- The AI uses **Minimax** to explore all possible future moves
- It picks the move that maximizes its own score while minimizing yours
- Human plays as `X`, AI plays as `O`

## How to Run

```bash
python tictactoe.py
```

No pip installs needed — uses only Python's built-in libraries.

## Sample Game

```
  1 | 2 | 3
 -----------
  4 | 5 | 6
 -----------
  7 | 8 | 9

Your move (1-9): 5

  X | 2 | 3
 -----------
  4 | O | 6
 -----------
  7 | 8 | 9
```
